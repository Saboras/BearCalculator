# 🐻 BearCalculator

A single-file web tool that works out troop **squad compositions** for the **Bear Trap event** in *Kingshot* — your own rally (a full squad) plus 6 capped joiner rallies, given your troops, hero/Marshal/Deployment buffs and target ratio.

## Use it

Open **[`index.html`](index.html)** in any browser. No install, no server, works offline — to share it, just send the HTML file (or host it on GitHub Pages).

## Highlights

- 9 squad sizes from base cap + Valora/Cassia + Marshal + Deployment buff.
- Own rally + 6 joiners with **Arch → Cav → Inf** backfill when archers run short.
- Joiner split modes: uniform · half & half (3+3) · thirds (2+2+2).
- Diagnostics: troops missing for a perfect ratio, leftover, total rally size, Personal Joiner Size.
